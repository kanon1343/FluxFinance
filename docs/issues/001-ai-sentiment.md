# [Feature] 銘柄別 AIニュース・センチメント分析の導入

## 概要
各銘柄に関連する最新の金融ニュースを取得し、LLM（大規模言語モデル）を用いてそのニュースが市場に与える影響（強気・弱気・中立）を分析し、ダッシュボード上に視覚的なバッジと一言要約を表示する機能を追加する。

## 背景・解決したい課題
現状のFluxFinanceダッシュボードは株価の現在値や過去の推移といった「数値」の提示に留まっており、価格変動の「理由」や「背景」を即座に把握することが難しい。ユーザーが自身でニュースサイトを巡回して要因を調べる手間が発生している。

## ユーザーストーリー
* **As a (〜として)**: 効率的に市場動向を把握したい個人投資家
* **I want to (〜したい)**: 銘柄ごとの最新ニュースのセンチメント（市場心理）と要約をダッシュボード上で一目で確認したい
* **So that (なぜなら)**: 時間をかけずに株価変動の要因を理解し、投資判断の材料にしたいから

## 技術的アプローチ・詳細
1. **APIルートの作成:** `src/app/api/sentiment/route.ts` を新規作成する。
2. **データ取得:** Yahoo Finance等のAPIを活用し、指定されたTickerの最新ニュース記事（見出し・概要）を取得する。
3. **LLM連携:** 取得したニューステキストをプロンプトに埋め込み、ローカルのOllama（Gemma）または外部APIに送信して、「Bullish / Bearish / Neutral」の判定と一言要約を生成させる。
4. **UIの更新:** `StockCard.tsx` および `ChartModal.tsx` を拡張し、取得したセンチメントバッジ（例: 🔥, ❄️, ➖）と要約文を表示するコンポーネントを追加する。

## 受け入れ条件 (Acceptance Criteria)
- [ ] `DEFAULT_TICKERS` に含まれる各銘柄の最新ニュースを取得できること。
- [ ] LLMを介してニュースのセンチメントが3段階（Bullish/Bearish/Neutral）で判定され、JSON形式で結果が返ること。
- [ ] ダッシュボード上のStockCardに、センチメントを示すバッジアイコンが正しく表示されること。
- [ ] ChartModal内に、LLMが生成したニュースの一言要約が日本語で表示されること。
- [ ] ニュース取得やLLM APIがエラーになった場合でも、画面全体がクラッシュせず、エラーメッセージまたはデフォルト状態が適切に表示されること。
