# [Feature] 自然言語ベースの「What-If」ポートフォリオシミュレーター

## 概要
ユーザーが自然言語で特定の仮想シナリオ（例：「もし来月テクノロジー株のバブルが弾けたら？」）を入力すると、LLMが過去の市場データやマクロ経済の相関関係を分析し、ダッシュボード上の各銘柄に与える影響やリスクを定性的・定量的にシミュレーションしてレポートする機能。

## 背景・解決したい課題
投資家は常に「将来の不確実なイベントが自分の資産にどう影響するか」を気にしているが、個人が複雑なマクロ要因を分析してポートフォリオへの影響を予測することは極めて困難である。従来のツールは過去のバックテストしか提供しておらず、仮定の未来に対するシミュレーションができない。

## ユーザーストーリー
* **As a (〜として)**: マクロ経済のイベントリスクに備えたい投資家
* **I want to (〜したい)**: 「もし金利が引き上げられたら」といったシナリオを自由記述で入力し、保有銘柄への影響予測を見たい
* **So that (なぜなら)**: リスクを事前に把握し、ポートフォリオのリバランスなどの対策を検討したいから

## 技術的アプローチ・詳細
1. **シナリオ入力UI:** ダッシュボード上部に「What-If シミュレーター」の検索・入力バーを追加。
2. **シナリオ解析エンジン:** 入力された自然言語のシナリオをLLMで解析し、各セクターや銘柄に与える影響度（スコアや増減予測率）を抽出する。
3. **データ合成:** `yahoo-finance2` のヒストリカルデータ（過去の類似イベント時のデータ）とLLMの推論結果を掛け合わせ、シミュレーション結果のデータ構造を生成。
4. **予測表示 UI:** `ChartModal` 等に、現状のチャートとは別枠で「シミュレーション予測線」を点線等で表示し、LLMによるリスク解説テキストを併記する。

## 受け入れ条件 (Acceptance Criteria)
- [ ] ユーザーがテキスト入力欄に任意のシナリオ（テキスト）を入力して実行できること。
- [ ] シナリオ実行後、ダッシュボードの各銘柄（StockCard）に予測される影響（例: 📉-5% 予想）が表示されること。
- [ ] LLMによって生成された、シナリオ全体の解説レポート（なぜそのように動くのかの根拠）がモーダル等で表示されること。
- [ ] 処理時間が長くかかることが想定されるため、シミュレーション中は適切なローディングUI（スケルトンやプログレスバー）が表示されること。
